# CS777_GroupProject## How to Run- Run the file main_s3.py    - Use s3://777-termproject/csv/flights_2019_full.csv as the non cleaned version of the data    - Use s3://777-termproject/processed/cleaned_preprocessed_2019/ as the cleaned version    ## Workflow- Everything runs from main_s3.py- Data is cleaned independently in data_cleaning.py and is called from main_s3.py- This data is then read in from the s3 bucket it was written to- Then the data is split, processed, scaled, and one hot encoded from data_cleaning.py- Afterwards, we create a basic logistic regression model in logistic_regression.py- Finally, we find all sorts of different metrics in evaluation.py
